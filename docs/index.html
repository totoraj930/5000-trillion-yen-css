使い方

0. フォントを設定しておく

このページではGoogle Fontsの「Noto Sans Japanese」を使用しています。

h1 {
	font-family: "Noto Sans Japanese", sans-serif;
	font-size: 2em;
}

1. CSSを読み込む

<link rel="stylesheet" href="5000-trillion-yen.css">

2. spanで囲む

<h1><span>5000兆円</span></h1>

3. classを付ける

<h1><span class="number-5000-i">5000兆円</span></h1>

3. title属性に同じ文字列をいれる

<h1><span class="number-5000-i" title="5000兆円">5000兆円</span></h1>

4. ほしい！
